# ShapeyShape
## Overview

**ShapeyShape**, a screensaver for Mac. 


## Technologies Used 

**1.** Objective-C

**2.** X-Code


## Prerequisites

1. X-Code on your Mac (get from App store)



## Usage / Features

Your Mac can recognize it as a screensaver and you can choose it under System Preferences >> Desktop & Screensaver.
